## 밤길 친구 : 안전 귀가를 위한 앱 개발 
### 안전 기반 경로 안내 서비스 
- 2020 WISET ICT 프로젝트 참여 작품
- 덕성여대 WISET 사업단 지원
- DATE : 200304 - 201031
- TOOL : Android Studio & Python & API
---
### 서비스 구현 
**1. 불안감 감소 장치**
  - 안전 지도 
  - 안심 경로 서비스 
  - 밤길 친구 서비스 
 
 **2. 실질적 도움 장치**
  - 자동 녹화 기능 
  - 비상 상황 알림
  - SOS 신고 기능
  ---
### 개발 
- 행정안전부 및 경찰청의 API 데이터를 지도상에 시각화
- 사용자에게 최적의 경로를 제시하기 위한 사용자 패턴 분석
- GPS 연계 학습을 통한 실시간 경로 안내 구현
- 자동 녹화,알림 발생 등의 기능을 수행할 자동화 시스템 구축
